The origdata folder contains the original data files for our analyses of ACS data.
This document describes the sources of these files.

cznames.dta
  A dataset created by the authors linking CZ codes to their names.
  
dorn/cw_puma2000_czone.dta
dorn/cw_puma2010_czone.dta
Dorn/cw_cty_czone.dta
  These are crosswalks from ACS PUMAs and counties to CZs. They were created by David Dorn. Further
  documentation is in the readme in the dorn folder.
  
housing/unix_hus.zip
  This is a zip file containing 4 SAS data sets, downloaded from Census Bureau web site,
  that are the household records from the 2018 5-year ACS file. URL is:
    https://www2.census.gov/programs-surveys/acs/data/pums/2018/5-Year/unix_hus.zip

rawacs/csv_pus_2018.zip
